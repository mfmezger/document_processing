## Installation
You need a version of Python 3.11.3 installed.

### Install Dependencies
```bash
    pip install poetry
    poetry install
```

### Run the OCR Demo

```bash
    poetry run python document_processing/main_ocr.py
```


### Run the Frontend

```bash
    poetry run streamlit run document_processing/frontend.py
```
